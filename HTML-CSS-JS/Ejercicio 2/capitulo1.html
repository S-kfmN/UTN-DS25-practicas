Capítulo 1: ¿Qué es HTML?

En este capítulo aprenderás qué es HTML, cómo funciona, y cómo escribir tu primera página web.

HTML significa Lenguaje de Marcado de Hipertexto. Es el lenguaje que usamos para crear páginas web.

Se escribe usando etiquetas, como por ejemplo html, head, title y body.

Una etiqueta abre con <nombre> y cierra con </nombre>. Por ejemplo: <p> y </p> para un párrafo.

HTML permite mostrar textos, imágenes, enlaces, listas y mucho más.

Las páginas HTML pueden abrirse con cualquier navegador web.
