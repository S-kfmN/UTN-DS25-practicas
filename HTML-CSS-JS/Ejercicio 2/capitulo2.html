Capítulo 2: Estructura básica de una página

Una página HTML contiene una cabecera, un cuerpo y muchas etiquetas que definen su contenido. Es importante respetar la sintaxis y las convenciones.

Una página HTML se compone de varias partes:

  • html: es la etiqueta principal de la página.
  • head: contiene el título, información y enlaces a estilos.
  • title: define el nombre que aparece en la pestaña del navegador.
  • body: es donde va el contenido visible para el usuario.
