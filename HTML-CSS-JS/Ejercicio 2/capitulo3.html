Capítulo 3: Enlaces y Listas

Los enlaces permiten conectar páginas. Las listas pueden ser ordenadas <ol> o no ordenadas <ul>.

En este capítulo vas a practicar cómo enlazar páginas internas y externas y cómo construir listas.

HTML permite agregar enlaces para navegar entre páginas. Se usa la etiqueta a y el atributo href.

Los enlaces pueden ser internos (a otra página del mismo sitio) o externos (a otra web).

También podemos crear listas para mostrar varios ítems:

  • Listas ordenadas: usan números.
  • Listas no ordenadas: usan viñetas.

Cada ítem de la lista se escribe con la etiqueta li.

Ejemplos de uso: lista de capítulos, lista de hobbies, pasos a seguir.
